# Multithreaded File Word Counter in C++

## 🚀 Overview
This project is a C++ based multithreaded application that counts the total number of words across multiple text files concurrently using `std::thread`.

It demonstrates core system-level programming concepts such as multithreading, synchronization, and file handling.

---

## 🎯 Features
- Multi-threaded file processing using C++ threads
- Concurrent execution for improved performance
- Thread-safe operations using mutex (`std::mutex`)
- Efficient word counting using file streams
- Error handling for file access

---

## 🛠️ Technologies Used
- C++
- STL (Standard Template Library)
- Multithreading (`std::thread`)
- Synchronization (`std::mutex`, `std::lock_guard`)
- File Handling (`ifstream`)
